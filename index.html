Get it on your phones
Step-by-step GitHub Pages setup

The goal: put this app on a free URL so both of you can open it on your phones anytime. GitHub Pages gives you a free link in about 3 minutes. No coding knowledge needed.

1
Create a free GitHub account
Go to github.com and sign up. It is completely free. Pick any username — it will appear in your app URL so make it something nice like ankishandsimar.
2
Create a new repository
Once logged in, click the green New button (or the + icon top right → New repository). Name it something like almost-melbourne. Set it to Public. Tick "Add a README file". Click Create repository.
3
Upload the index.html file
Inside your new repository, click Add file → Upload files. Drag and drop the index.html file you downloaded from here. Scroll down and click Commit changes.
4
Turn on GitHub Pages
In your repository, click Settings (top tab row). Scroll down the left sidebar to Pages. Under "Source", select Deploy from a branch. Set branch to main and folder to / (root). Click Save.
5
Wait 60 seconds, then open your URL
GitHub will give you a URL that looks like this:
https://YOUR-USERNAME.github.io/almost-melbourne
It takes about 60 seconds to go live after you save. Refresh the Pages settings page and the link will appear at the top. Send it to Simar. Bookmark it on both phones. Done.
6
Add it to your home screen (optional but recommended)
On iPhone: Open the URL in Safari → tap the Share icon → tap "Add to Home Screen" → tap Add.

On Android: Open in Chrome → tap the three dots menu → tap "Add to Home Screen".

It will appear like an app icon on your home screen.

To update the app later (e.g. add a new photo or edit questions): go back to your GitHub repository, click on index.html, click the pencil edit icon, make changes, and click Commit. The live site updates automatically within a minute.
